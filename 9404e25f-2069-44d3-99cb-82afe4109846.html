Description

REPTILAND is a task-based FNAF fangame where you play as a zookeeper completing duties. You get to do 7 different tasks in total, 2 being main and the others optional, such as heating the rooms, feeding the animals, carrying boxes, and more.

This game also features Game Jolt API integration with guest scoring (meaning that you don't need to be logged in to participate), where you get to compete with developers and other players' times!

The game is still currently in development, follow the game page or join the community to stay in put with any news!
