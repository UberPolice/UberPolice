Description

A game about a coin block, some clicking, and a choice.

Created for (#LDJAM) Ludum Dare 33: You are the Monster
Jam Page: http://ludumdare.com/compo/ludum-dare-33/?action=preview&uid=39687

If clicking isn't your thing, some hotkeys:
Space - Triggers the Coin Block
s - Opens the Shop
n - Mutes Sound
enter - Move the dialog along
mouse wheel/dragging - Scroll through the shop menu

Please let me know what you think in the comments section! Also, this is my first (Unity)WebGL game, so if you find any bugs (memory errors, game slows down or crashes, other oddities) please leave a comment and I'll look into it. I'm pretty sure I got everything, but you never know.

I'm including Coin Block Clicker in #gjfest because it provides a unique story and a new take on the clicker genre.

Enjoy, and thanks for playing!
